Ladda Bootstrap

A UI concept by @hakimel which merges loading indicators into the action that invoked them. Primarily intended for use with forms where it gives users immediate feedback upon submit rather than leaving them wondering while the browser does its thing. This project is used by @hakimel slid.es and by @msurguy on Bootsnipp.
